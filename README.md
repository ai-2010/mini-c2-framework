# Packet Sniffer

Network paketlarni ushlash va tahlil qilish.

Fayllar:
- `sniffer.py`: sniffing boshlovchi
- `packet_parser.py`: paketdan source/dest/protocol/payload chiqarish
- `filters.py`: filtr funksiyalari (TCP, UDP, IP)

Ishlatish:
```
python sniffer.py
```

Eslatma: `scapy` o‘rnatilgan bo‘lishi kerak.
